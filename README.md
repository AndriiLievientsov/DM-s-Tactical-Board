# 🎲 DM's Tactical Board v2

**DM's Tactical Board v2** — это легковесный, быстрый и отзывчивый веб-интерфейс (Virtual Tabletop / VTT), разработанный специально для мастеров настольных ролевых игр (D&D, Pathfinder и др.). Приложение позволяет визуализировать сражения, управлять инициативой и вести учет состояния персонажей прямо в браузере без тяжелых сторонних платформ.

Интерфейс выполнен в эстетичном темном киберпанк/фэнтези стиле с акцентными цветами, не утомляющими глаза во время длительных игровых сессий.

---

## ✨ Основные возможности (Features)

### 🗺️ Управление картами и локациями

* **Мульти-локации:** Создавайте дерево игровых зон («📍 Локации») и мгновенно переключайтесь между ними.
* **Интерактивный холст:** Загружайте свои карты боевых столкновений простым перетаскиванием (Drag-and-Drop).
* **Умная навигация:** Плавное масштабирование (Zoom колесиком мыши от 20% до 500%), панорамирование (клик колесиком) и полноэкранный режим.
* **Динамические линейки координат:** По краям холста расположены буквенно-цифровые линейки (A-Z, 1-N) для быстрой навигации и целеуказания.

### 🌫️ Инновационная система «Тумана Войны» (Fog of War)

* **Три режима нанесения:** Рисуйте скрытые зоны произвольной **Кистью**, выстраивайте сложные геометрии через **Полигоны** (по кликам) или быстро закрывайте области **Прямоугольником**.
* **Настройка плотности:** Регулируйте прозрачность тумана от полупрозрачной дымки до абсолютной тьмы.
* **Режим Мастера (DM View):** Скрытые зоны тумана остаются видимыми для вас в виде аккуратных пунктирных областей, которые можно в любой момент показать игрокам или удалить через контекстное меню.

### 🪙 Токены и кастомизация персонажей

* **Библиотека ассетов:** Встроенный менеджер файлов («📁 Библиотека») с поддержкой папок и Drag-and-Drop загрузки картинок для токенов.
* **Продвинутое контекстное меню персонажей:**
* Быстрое изменение имени и текущих хитпоинтов (HP).
* Изменение размера токена по сетке: от *Крошечного (0.5x)* до *Гигантского (4x)*.
* Свободный поворот токенов по направлениям (шаг 90° / 180°).
* **Трекер статусов:** Наложение визуальных эффектов-иконок прямо на токен (🔥 Горит, 🤢 Отравлен, 🕶️ Ослеплен, 🧎 Сбит с ног, 🛡️ Концентрация и др.).



### ⚔️ Интегрированный боевой трекер (Initiative Tracker)

* Автоматический импорт всех токенов с карты в очередь инициативы в один клик.
* Интерактивное управление здоровьем прямо из сайдбара с помощью быстрых кнопок (`-5`, `-1`, `+1`, `+5`).
* Сортировка по значению инициативы, ведение счета раундов и автоматическая подсветка текущего ходящего токена на карте мягким зеленым пульсирующим свечением.

### 🛠️ Инструменты Мастера

* **Векторное рисование и Ластик:** Рисуйте схемы, помечайте ловушки и стирайте ненужные элементы на ходу.
* **Тактическая сетка:** Переключаемая сетка (белая/черная) с плавной регулировкой прозрачности.
* **Линейка расстояний:** Инструмент измерения, переводящий пиксели в игровые футы (шаг 5 фт.).
* **Система пингов (Сигналы):** `Shift + ЛКМ` создает на карте красивые расходящиеся анимированные круги для привлечения внимания игроков.
* **Горячие клавиши и буфер обмена:** Поддержка отмены/повтора действий (Undo/Redo), группового выделения («Маркиза») и классических комбинаций `Ctrl+C` / `Ctrl+V` для клонирования токенов.
* **Полная автономность:** Быстрое сохранение всей сессии и кампании в один компактный `.json` файл и мгновенная загрузка.

## 🚀 Быстрый старт (Quick Start)

Приложение является **Single Page Application (SPA)** и написано на чистом JavaScript (Vanilla JS) без использования тяжелых фреймворков и внешних зависимостей.

1. Склонируйте репозиторий или скачайте код проекта.
2. Откройте файл `index.html` в любом современном браузере.
3. Всё готово к игре! Приложение работает полностью локально и не требует сервера.

---

## 🎨 Стек технологий (Tech Stack)

* **HTML5 Canvas:** Высокопроизводительный рендеринг слоев карты, токенов, рисунков и тумана войны.
* **CSS3:** Современные CSS Variables, адаптивная Flex/Grid верстка сайдбара и кастомный стилизованный скроллбар.
* **Vanilla JavaScript:** Собственная событийно-ориентированная архитектура (Event Emitter), кэширование изображений (`Map`), и алгоритмы математического просчета коллизий (Ray-casting для полигонального тумана).
